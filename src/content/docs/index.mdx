## Why AgentLoop?

AgentLoop wraps the **pi coding agent** and adds everything needed to run it in production: persistent memory, human-in-the-loop safety gates, session management, and multi-client support — all accessible via a Unix socket JSON-RPC API.

<CardGrid>
  <Card title="Persistent Memory" icon="pencil">
    User profiles, conversation history, three compaction strategies, and prompt caching. The agent remembers your preferences, projects, and past interactions across sessions.
  </Card>
  <Card title="HITL Safety" icon="warning">
    Human approval gates for dangerous operations — docker commands, git push, destructive file operations, and network requests. Works in CLI and Slack.
  </Card>
  <Card title="Multi-Client Support" icon="laptop">
    CLI, Slack, and future web UI — all connected via Unix socket JSON-RPC. One server, multiple clients, shared state and memory.
  </Card>
</CardGrid>

## Quick Start

```bash
# Build both binaries
go build -o agentloop-server ./cmd/agentloop-server
go build -o agentloop ./cmd/agentloop

# Start the server
./agentloop-server &

# Run your first task
./agentloop "list files in the current directory"
```
